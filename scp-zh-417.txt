**項目編號：**SCP-ZH-417

**項目等級：**Safe

**特殊收容措施：**SCP-ZH-417應被持續收容於Site-ZH-11的417號收容盒內，並保持電池槽淨空。417號收容盒外的鑲嵌小盒中應放有六枚符合SCP-ZH-417需求尺寸的鈕扣電池。

使用SCP-ZH-417時請勿直接向人員眼球照射，使用者應配戴裝有UV400鏡片的太陽眼鏡、長袖衣物或可遮蔽大面積身軀的衣物、面具或矇住頭部的布料，以及手套和襪子，以上所述的衣物布料材質皆以不透光之日常衣物為準。

受到SCP-ZH-417照射的食物應在實驗結束後燒毀。

[[include component:image-block name=https://i.imgur.com/9vibjdE.jpeg maxwidth=760&fidelity=grand|caption=SCP-ZH-417本體]]


**描述：**SCP-ZH-417是一個長135毫米、寬19毫米、厚13毫米的筆燈，以AG-13型號或相同尺寸的鈕扣電池供電。SCP-ZH-417來自2024/11/6於台南市中西區由基金會破獲的帷幕內新興邪教群體「燈光」，透過訊問得知SCP-ZH-417為其教團用以控制教徒的核心物件。

SCP-ZH-417有兩項異常效應，第一為其發出光線對人類的吸引性、振奮性、成癮性，目視SCP-ZH-417發射在物體上的光點或被光點接觸皮膚皆會受到異常效應影響，避免受到目視異常效應的方法為戴上裝有UV400鏡片的太陽眼鏡，其餘則以不透光衣物遮擋身軀；第二項為第一項異常效應的延伸，受光線照射過的食物將帶有相同性質，但強度比直接目視或接觸光點弱上許多，此處食物的界定範圍為可食用物。

受到SCP-ZH-417異常效應影響的人類將感受到放鬆及愉悅，並促使大腦釋放多巴胺，人類在接觸SCP-ZH-417的光點後有近99%的機率對其產生心理依賴，在停止目視或接觸SCP-ZH-417發射的光點後，受影響人類將產生戒斷反應，在被控制的情況下將以自我傷害的方式要脅周遭的人類讓自己能夠重新目視、接觸SCP-ZH-417發射的光點，或品嘗受光線照射過的食物，幫助自己接觸能使其重新體驗的人物。

>> = **【遺物相關紀錄：SCP-ZH-417】**

>> **製造時間：**1999/12/1
>> 
>> **作品標題：**希望牌祈光食鹽
>> 
>> **必須素材：**
>> * 繁廣化工 LLA-76鋁合金
>> * 繁廣化工 LO-66微小型燈泡
>> * 適量的愛人骨灰

>> **摘要：**
>> 
>> 人類是需要希望才能活下去的，但生命無常，絕望更常會發生在我們的身旁，再陷入絕望之際，哪怕是一點點希望，都能夠成為被絕望打擊後的生存動力，這隻筆燈融合了我數年掌握的奇異技術，將帶給眾人希望的燈光。
>> 
>> 我們要將使人絕望的東西磨入特別的鋁合金中，這是繁廣化工停產多年的產品，能夠使原料的力量轉向，而這個型號的燈泡則擁有增幅效果，同樣是停產多年的產品，我花了好大的力氣才找到，我的工匠意念也會滲入到這個作品之中，這個材料的不確定性與獨特性將使每一份作品都獨一無二，希望的燈光將照耀憂鬱者與陷入低谷者。
>> 
>> 精神上的食糧也不足以將人帶出低谷，這個燈光將會使人能夠配著其他食物將希望吃下，擁有健康的身體才容易擁有健康的精神。
>> 
>> **意圖：**
>> 
>> 我是一位失格的丈夫，我沒有堅決地要我的妻子留在家中，我遲疑了，他被我醉酒的長女女婿撞死了，故意的，原本他要撞死我，我的長女在喪母後依舊很愛他，他們去了台北，他偶爾會回來看我，但身上總是有瘀傷，臉色看起來非常糟糕，也不怎麼吃東西，瘦到被海風一吹就會被刮到深山裡面。
>> 
>> 我是一位不稱職的父親，憂鬱症先後纏上了我的兩位女兒，我讓兒子趕緊離開台灣，永遠不要回來，我把他罵得一文不值，他一氣之下在長女葬禮後移民去美國，現在我只需要好好照顧我的次女。
>> 
>> 次女沒有出嫁，一直陪著我和妻子，他同樣很愛他的兄姐，但繼承了我的猶豫，所以和我一樣內疚沒有及時將他的姐姐和母親拉回來，所以他變得開始像他的姐姐的樣子，我很心疼，瘦得不成人樣。
>> 
>> 這個作品不只是我的金屬加工技藝的展現，還同時是拯救孩子的營養品。
