+++ **特殊收容措施**
Site-ZH-16收容一區的597號收容櫃中應收容兩種尺寸的SCP-ZH-597各三片，其餘被回收的SCP-ZH-597於2024年四月後皆被送往[[[SCP-ZH-234 | Site-ZH-16 汙染消解部]]]進行銷毀。

若發現任何被公布在帷幕外的SCP-ZH-597相關資訊，則由MTF-破軍-09 「七星資訊」進行掩飾與誤導，MTF-暖爐-09 「野火燒盡」進行刪除與清理。

+++ **描述**
SCP-ZH-597是由繁廣化工生產的地磚，有兩種尺寸，一種是六十公分乘六十公分，另一種則是八十公分乘八十公分。該磁磚生產線已由基金會清查與銷毀，SCP-ZH-597的相關產品於2023年後停產，回收作業暫定2026年結束。基金會在清查生產線前，生產SCP-ZH-597所用的多項機器便受到提前破壞，在確認進入生產鏈前的製造原料堆並無異常後，SCP-ZH-597的異常特性被推測來自於製造過程中機器所添加的其他原料。後續圍捕的相關繁廣化工人員皆採取自殺的方式避免問話，SCP-ZH-597的相關研究進度受阻。

SCP-ZH-597的異常效應要滿足兩個條件才會發生，第一是SCP-ZH-597需要被鋪設，第二是當下發生返潮。當條件被滿足，每片SCP-ZH-597都會出現三到四雙肉眼看不見的手，這些手可以透過紅外線熱像儀發現，尺寸與成年亞洲男性平均相仿，被編號為SCP-ZH-597-A。

SCP-ZH-597-A會輔助通行者，並保持通行者的平衡以及速度，使其與走在正常的乾燥地面相同，通行物的移動亦同。若通行者意外跌倒，SCP-ZH-597-A會彼此相握，組成一個緩衝網接住通行者，通行者的判定廣泛，各項測試顯示只要是活著的動物皆在此範疇之內。

放置物會被SCP-ZH-597-A略微抬起以避免接觸潮濕的地面而受潮，抬起的高度通常為兩到五毫米，若SCP-ZH-597上已有防潮布或棧板等物，則此效應不會發生。此效應較特殊的案例是抹布與拖把，抹布或拖把若在返潮發生的情況下被放在SCP-ZH-597之上，SCP-ZH-597-A會利用拖把或抹布將所在磁磚的水滴擦乾。

SCP-ZH-597-A無法離開自己的所屬的SCP-ZH-597，但當多片SCP-ZH-597被拼在一起，鄰近的SCP-ZH-597-A可以互相支援，種種跡象顯示SCP-ZH-597-A本身應擁有一定的智慧，但對此相關的研究皆無法確認該智慧的來源為何。

若SCP-ZH-597被分割，則面積少於原尺寸一半的部分失去異常性，故以硬物敲擊SCP-ZH-597使其支離破碎是最有效率的無效化方式。

SCP-ZH-597作為壁磚使用的後果詳見【附錄：壁磚回收事件】。

>> 
>> 回收隊伍在抵達後亦受到同樣效應影響，原因是SCP-ZH-597被當成壁磚鋪設。在使用槌子破壞SCP-ZH-597後成功救下當事人，當事人被帶往基金會前台旅館休息，以及接受訪談與記憶刪除。
>> 
>> **備註：**當事人屋中的壁磚已由SCP-ZH-597贗品重新鋪製而成。

>> 根據各方錄影與走廊監視器所記錄，事發當日下午四點，撞擊聲變得頻繁，事發屋的門發出明顯的震動，下午四點十分時走廊已有十位同層住戶、五位不同層住戶聚集。下午四點十七分，事發屋的門被撞開，一具人體被從中拋出，同時引起驚慌。經過後續相驗，確認該人體為事發屋租客，死亡時間約五日，死於心臟病。
>> 
>> 基金會於當日介入調查，並對資訊進行壓制、誤導，以及刪除。後續發現事發屋於去年底有進行裝修，裝修過程中使用到流入市面的SCP-ZH-597，異常效應可以透過該租客在屋中設置的鏡頭發現，其死亡後受到SCP-ZH-597-A抬起，並被用來撞擊牆壁與門。
>> 
>> 目前推測由於該屋對外窗周遭並無鋪設SCP-ZH-597，靠近正門的SCP-ZH-597-A是以試錯的方式找到門，並將該租客拋出。若該段時日未發生返潮，預估需要將近半個月才能發現租客屍體。
>> 
>> 值得一提的是根據事發前與事發當時的影像，可以透過撞擊聲的頻率與方式發現部分時段的敲擊是以摩斯密碼的形式在傳遞消息，經過破譯後，將原本屬於英文訊息翻譯可以得到：「我有訊息傳送」、「此處有屍體」、「訊息結束」。
>> 
>> **備註：**已對屋主與裝修公司進行調查，兩者並無帷幕內知識，應是誤用SCP-ZH-597。
