[[collapsible show="查閱舊版收容措施＋" hide="close"hideLocation="both"]]
由於該項目喜好親近未成年者（女性）的特性，當前由物流部主任Dr. Sugoi[[footnote]]耶----我有妹妹了！[[/footnote]]、Willow 特工[[footnote]]我要抱她玩飛高高！[[/footnote]]、研究員L.What[[footnote]]對於我性別上的歸類，都沒人有半點懷疑嗎？[[/footnote]]、Dr.X[[footnote]]這是想吵架嗎？[[/footnote]]等在外觀和視覺年齡上較不易被該項目提防的人員輪流監管。在基金會指定的時間（20██年██月██日）到期前，上述人員與SCP-ZH-705-A進行正面交流（包括但不限於交談、遊戲、朗讀故事、進行符合其年齡的兒童教育行為）並協助其適應基金會的生活。
[[/collapsible]]

**特殊收容措施：**

當前SCP-ZH-705收容於異常醫學部門研究主任Dr.Siren的辦公室，任何需要該項目進行的異常實驗必須經過Dr.Siren的授權許可，且在實驗結束後。所有SCP-ZH-705-A要求的物品以燃燒紙紮品的方式傳送，為此負責製作紙紮品的業務已交由基金會前台的殯葬公司負責。食品與玩具類則交由Dr.Siren轉送。

**描述：**SCP-ZH-705是一張邊長30公分，高度20公分，以竹藤編製的傳統四角椅凳，當該項目置於20歲以下的對象時，一位年齡約3～4歲的女性兒童（指定為SCP-ZH-705-A）會出現在對象面前。

SCP-ZH-705-A屬於臺灣民間降靈儀式過程中出現的靈體異常，在與其進行溝通後，已確認該異常為民間傳說中的「椅仔姑」。當前與SCP-ZH-705-A的溝通方式，除了透過Dr.Siren轉述，將以一臺1980生產的老式收音機作為傳達訊息的媒介。

以下為SCP-ZH-752-A曾要求的非紙紮物品清單：

> * 統一布丁（大）
> * 森永牛奶糖（12粒裝）
> * 巧克力（偏好金莎和大波露）
> * 旺旺仙貝（小包裝）
> * 嗨啾軟糖（偏好青蘋果、柳橙、芒果、葡萄、荔枝和水蜜桃）  
> * 高度30公分的彼得兔
> * 萬花筒
> * 布丁狗絨毛布偶
> * 風車（四瓣和六瓣）
> * 液態沙漏玩具（壓克力製）


在進入基金會後，SCP-ZH-705-A之休謨指數一直處於穩定的50～60間，雖然已能呈現肉眼可見的形體。但僅被允許於Dr.Siren的辦公室內活動，當Dr.Siren外出時，SCP-ZH-705-A不得離開前者身邊2公尺內。

雖然SCP-ZH-705-A的個性相當害羞，但在與Dr.Siren外出時會出現分享零食給遇見的對象（無論是否認識），顯示SCP-ZH-705-A對成年人的恐懼有明顯減少，而基金會人員可以決定是否收下。

> **程██：**呵……是起笑[[footnote]]台語的發瘋[[/footnote]]吧？所以我才不想讓別人知道……是不是很驚訝我到現在還沒被送到精神病院？（苦笑）
> 
> **特工 賓果：**我不怕，程同學，我真的不怕……這個過一會兒就消了。我知道妳這陣子很難熬，妳只是因為這些糟心事產生了創傷，再加上家庭環境造成妳的自我懷疑，但妳絕對不是神經病。
> 
> **程██：**那你覺得有病的是誰？
> 
> **特工 賓果：**我覺得是那些搞不清楚狀況，只會躲在螢幕後打鍵盤的[已刪除]，真正有病的人根本不會意識到自己有病，當然也就不會分辨是非對錯了。
> 
> **程██：**謝謝你的安慰喔。（苦笑）
> 
> **特工 賓果：**不客氣，好啦，家訪就到這裡結束，照約定，妳不會再見到我了。喔對了，這給妳。
> 
> **程██：**三明治&咖啡樂園？（Sanwich＆Coffee Paradise）這什麼鬼？
> 
> **特工 賓果：**喔，這是我朋友自己新開的店，主要是賣簡餐和下午茶。妳現在應該是能打工的年紀了吧？他想找些工讀生，如果妳有經濟上的困難，就去那裡應徵看看。
> 
> **程██：** 喔，謝謝。不過我已經有酒吧的工作了，不過我會留著以後再說。
> 
> **<紀錄結束>**
> 
[[/collapsible]]


[[collapsible show="附錄ZH-705-4：第二次訪談" hide="close"hideLocation="both"]]
> = **[[size 1.5em]]第二次訪談[[/size]]**
> 
> --------
> 
> **日期：**20██年██月██號
> 
> **採訪者：**Dr.Cocoa
> 
> **受訪者：** SCP-ZH-705-A
> 
> **前言：**由於當前SCP-ZH-705-A只肯與程██交談，基金會聯絡上程██，在程██與其溝通後，研究團隊得以在無須進行降乩儀式的前提下，成功召喚SCP-ZH-705-A，由於後者的性質無法進行錄影，僅有音檔記錄。
> 
> **＜開始紀錄＞**
> 
> 
> **Dr.Cocoa：**早安，小妹妹。
> 
> **SCP-ZH-705-A：** 你好！（用活潑的語氣打招呼）
> 
> **Dr.Cocoa：**妳在這裡一段時間了，有沒有什麼不習慣的地方？
> 
> **SCP-ZH-705-A：**不會，可是你們為什麼要把我帶來這裡？姐姐呢？姐姐在哪裡？
> 
> **Dr.Cocoa：**什麼姐姐？
> 
> **SCP-ZH-705-A：**程██姐姐啊！我想回家，我要跟姐姐一起玩。
> 
> **Dr.Cocoa：**姐姐現在在外面，但是妳要先回答一些問題才能去找她喔。
> 
> **SCP-ZH-705-A：**會很久嗎？
> 
> **Dr.Cocoa：**不會，首先，能告訴我妳叫什麼名字嗎？
> 
> **SCP-ZH-705-A：**我叫小滿！二十四節氣的小滿！是姐姐幫我取的唷！（開心）
> 
> **Dr.Cocoa：**那妳真正的名字呢？
> 
> **SCP-ZH-705-A：**不知道，嫂嫂……（逐漸小聲）……嫂嫂都叫我喂和剋星仔（掃把星），我不記得了。
> 
> **Dr.Cocoa：**不記得也沒關係，那我也可以叫妳小滿嗎？
> 
> **SCP-ZH-705-A：**可以。
> 
> **Dr.Cocoa：**妳還記得以前的事嗎？就是妳和妳阿兄嫂嫂一起生活的那段日子裡（SCP-ZH-705-A在提到嫂嫂時顫抖了下），妳最後記得的事是什麼？
> 
> **SCP-ZH-705-A：**我記得……那天很冷，嫂嫂……嫂嫂說要煮飯，要我生火燒柴……然後，因為風很大，我要顧爐火，不能讓火熄掉，可是我好累、好想睡，我只是閉上眼睛睡一下下，醒來就在別的地方了。
> 
> **Dr.Cocoa：**睡醒之後就在姐姐家了嗎？
> 
> **SCP-ZH-705-A：**嗯，那個時候姐姐和三個不認識的姐姐（指廖██、趙█、蘇██）在好多大鐵盒的地方（指停車場），抓著椅子在玩，我也想玩，然後我發現自己可以飛好高好高，我就飛到椅子上，想讓她們看到我。
> 
> **Dr.Cocoa：**那她們有看到妳嗎？
> 
> **SCP-ZH-705-A：**沒有，她們都在講些聽不懂的話，我只好敲她們放的奇怪桶子，可是除了姐姐，她們還是看不到我。
> 
> **Dr.Cocoa：**後來呢？
> 
> **SCP-ZH-705-A：**我用力搖晃椅子，結果她們開始尖叫，其中一個姐姐生氣了，說要把我嫂嫂叫來，嫂嫂只要生氣就會打我，我就躲起來了。
> 
> **Dr.Cocoa：**最後只剩下妳跟姐姐了是嗎？
> 
> **SCP-ZH-705-A：**對啊，姐姐對我很好喔，而且她認識好多神明喔！有關公、土地公、城隍爺還有地藏王菩薩。她說只要我乖乖的，嫂嫂就找不到我，可是我該回家了。
> 
> **Dr.Cocoa：**那妳想回家嗎？
> 
> **SCP-ZH-705-A：**不想……可是不回家，嫂嫂會生氣。
> 
> **Dr.Cocoa：**如果妳喜歡這裡就叫姐姐帶妳來吧，在這裡不用做家事，妳嫂嫂他們也找不到妳，可是妳要乖乖的喔！
> 
> **SCP-ZH-705-A：**耶咿----（歡呼）那我可以找姐姐了嗎？
> 
> **Dr.Cocoa：**抱歉，我們有話要跟姐姐說，可以再等一下嗎？
> 
> **SCP-ZH-705-A：**喔，好吧。
> 
> 
> **<紀錄結束>**
> 
> **結語**：在結束和SCP-ZH-705-A的談話，基金會與程██達成協議，由前者安排能短暫安置的地方（僅程██與基金會指定的人員知情），程██將被列入觀察名單，若確定不構成威脅，未來將進一步列入基金會培訓名單，並對SCP-ZH-705重新進行收容。
>>
>> 說真的，這次事件發展很有意思，要知道我對靈媒、乩童這類稱呼是跟騙子劃上等號的（而且那根本不算職業），如果這孩子真的能和死者溝通，那的確有研究的價值，畢竟我們這邊確實蠻常在死人的。
>> [[>]]

>> [[/>]]
> 
>> 同意，一個不具異常性質的人類，可以透過非科學技術得知基金會的存在，而這樣的人可能還不只一個，如果可以尋找並透過所謂乩身，與其附身的神靈取得聯繫甚至進行合作也不是壞事。
>> [[>]]

[[collapsible show="附錄ZH-705-5：音檔記錄" hide="close"hideLocation="both"]]
在訪談結束後，潛伏在警局的特工將該項目調包並送到基金會，約兩週後，程██也寄了一個錄音檔包裹送到Site-ZH-██前臺。

>> 哈囉？……哈囉，有在錄嗎？很好，先打個招呼吧。
>> 
>> 嗯……MIB的各位你們好。抱歉這樣稱呼你們，我當然知道你們不叫這個名字，但讓我暫時這樣叫吧。
>> 
>> 相信你們應該知道我家裡是做什麼的了，所以不要拿我跟外面那群騙財騙色的下三濫相提並論，我會很不爽。
>> 
>> 我也相信你們應該從那位自稱「校園輔導人員」的人知道，我跟他發生的那段靈異小插曲了。
>> 
>> 沒錯，我看得到鬼，也看得到神明。這個能力從我出生就靈魂綁定[[footnote]]電玩術語，來自知名電玩遊戲魔獸世界內的設定，當某件裝備有著「靈魂綁定」這個屬性，代表該裝備將永遠綁定在「撿起它的人」，無法交易。[[/footnote]]了。至於我是如何得知你們存在，可以從我剛才說的話二選一，或者等哪天我們能夠面對面交流，我親自跟你們說。
>> 
>> 好了，回到正題，現在在我身邊的小妹妹確實是民間傳說的「椅仔姑」，你們未來想找祂占卜可以，但不要把祂算的結果當一回事，畢竟你們真的指望一個**死於數百年前的三歲小孩，知道哪支股票會漲、樂透彩開幾號、誰會當選總統這種事嗎？**
>> 
>> 不過也別擔心，我已經先幫你們試過了，祂可以算天氣、算感情、尋人尋物、警示危險、擋煞，這些小事是真的準。
>> 
>> 那麼問題來了，以宗教角度來說，一個在沒金身、沒香火甚至沒修為的情況下，早該在死後去陰間報到或是要被鬼差帶走的陰魂，為何還能存在於人間上百年？僅僅簡單的幾張民俗文獻就能讓祂留下嗎？我不認為，祂在這些記錄中連名字都沒有。而且照我們這行的規矩，這種滯留在人間的陰魂，要嘛超渡祂，要嘛滅了祂。因為祂的理智會隨著時間漸漸磨滅，最後變成無差別傷人的厲鬼，這對人間和祂自身都不是好事。可是這孩子卻對祂生前的事還有點記憶，我只是講到掃把這個詞，祂就消失個幾分鐘。祂還記得自己害怕什麼，為何害怕。
>> 
>> （嘖嘴聲）……該死，我喝個水，說話說太多，口好渴。
>> 
>> （走路聲＋喝水聲）
>> 
>> 好，我回來了，剛講到哪……總之我的意思是，我希望你們照顧祂。
>> 
>> 其實應該要說安置才對，因為祂不能留在我們家，我阿公也有陰陽眼，只是我能看到的拍咪啊[[footnote]]台語的髒東西，也可指亡靈。[[/footnote]]比他多，而我阿公不會因為祂是小孩就接納祂。可祂會出現在這裡，我也有一部份責任，我也不想祂落到哪個邪教巫師手裡。想來想去，也只好拜託你們了。
>> 
>> 你們要是同意的話就回覆吧，如果得要談條件才能接受的話，我們見面再談。
>> 
>> （錄音結束）
