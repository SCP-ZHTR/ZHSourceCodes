篇名︰**SCP-ZH-267 - 有緣有故**

本篇文章投稿2024夏季競賽《元素》

//**業力 蓮華 三界火宅**//

作者 : [[*user YONGZUN]]

[[include info:end]]
[[/=]]


**項目編號：**SCP-ZH-267

**項目等級：**Safe

**特殊收容措施：**SCP-ZH-267應被收容於Site-ZH-16收容一區內的267號收容間，有鑑於個體長度與個體數量，該收容間應裝設掛勾以利整理，並設計傾斜的防火材質地面，以利爆裂事件後的清掃工作。

爆裂事件發生後，事發的該SCP-ZH-267個體粉末應依照奧秘消解手冊第三版中的無害級異常產物程序處理，而事後探訪SCP-ZH-267-A發現的SCP-ZH-267個體應被收容進267號收容間，如果該地點兩次事件中沒有新的SCP-ZH-267個體出現，本項目研究主任Dr. Ming得裁決是否派遣人員主動進入SCP-ZH-267-B取得新的SCP-ZH-267。

若無本項目研究主任裁決，不得進入SCP-ZH-267-B。

**描述：**SCP-ZH-267是一類異常佛珠串，珠子的材質未統一，自木質至金屬質皆有，珠子的數量自18顆至108顆皆有，極少數為1080顆。SCP-ZH-267的出現推測至少在西元1968年即有所跡，有論者認為可以向上追溯至西元1938年，由於SCP-ZH-267的異常效應，目前的流通廣度極大，但極不容易被非帷幕內人士或非異常人形發現異常，在帷幕外仍被當作一般的佛珠串認知及處理。

本項目被基金會發現並收容的時間點為西元1992年，由陳默廉 博士發現並記錄在案，於Site-ZH-16建成後即內建267號收容間，並將當下蒐集到的108串SCP-ZH-267個體放入該收容間，此後收容間內的SCP-ZH-267個體便在108串上下浮動，歷史最低個體數量為18串。本項目負責人於西元2008年因陳默廉 博士的退休而由Dr. Ming繼承。

SCP-ZH-267個體的異常效應發生在持有人或持有團體將面臨災難前，自身將會爆裂，以阿基瓦輻射觀測器可以測量到個體當下的爆裂將呈現出特別的「福至圖型」[[footnote]]《阿基瓦輻射分佈專論：福至圖型與禍止圖型》；呂大郎 博士，〈崇高神學季刊〉第31卷第四期，(1986年10月)，112頁~119頁[[/footnote]]，使持有人或持有團體規避大部分或是全部災難損害，而珠子與中間的線四散後將自燃成粉，截至2024年，基金會已經透過SCP-ZH-267規避至少五次毀滅危機，以及三十次的極大損害，但SCP-ZH-267不一定在每次災難前都會作用，其中似乎有難以觀測的篩選機制，此機制為近年來SCP-ZH-267的主要研究方向。

SCP-ZH-267-A是位於新北市深坑區的一間廢棄宮廟，是目前已知的唯二SCP-ZH-267產地，其不定時將在其內部置中且唯一的香爐中出現，並以新的香灰覆蓋，但目前仍未發現來此祭拜的香客，雖然不穩定性較高，但相對低廉的成本使其成為基金會蒐集SCP-ZH-267的首選地點。SCP-ZH-267-A與周遭土地的原持有者已經死亡，該區域於2000年已成為國有地，該地目前已由民異署管轄，並與蘇陳彭開發有限公司[[footnote]]基金會前台公司[[/footnote]]締結50年的行政契約，由於未知原因，民異署對SCP-ZH-267展現抗拒態度。

SCP-ZH-267-B是SCP-ZH-267-A南方的一處小型隧道，推測是日治時期挖掘的防空洞，內部最深處有十八尊未損壞不知名金漆佛像，經探測發現內部為木質結構，以及放在正中間，一尊無眼、無耳、無鼻、無嘴的無金漆木質佛像，每尊佛像前方都有小香爐，小香爐中間則有一鼎大香爐。

根據經驗，無論進入SCP-ZH-267-B的行動需要多少人員，一個人員即可從大香爐的香灰下取出SCP-ZH-267，當人員從SCP-ZH-267-B取出SCP-ZH-267，會在金漆佛像前看見與取出項目相同數量的祭拜用香，木質佛像前則會有無法以任何技術識別身分的人眼、人鼻、人耳、人皮、人舌，數量也與取出項目數量相同。僅取出SCP-ZH-267後，SCP-ZH-267-B內部將發生高溫事件，屆時內部的溫度可以上升至七十二攝氏度。

若將出現之人體組織帶出，則高溫事件將會發生在該組織所在地，若僅取部分人體組織離開，則會在兩地出現高溫事件。//(2018/10/9編輯)//

> = **【附錄：SCP-ZH-267-B封鎖緣由】**
> 
>> **撰寫者：**陳默廉 博士
>> 
>> **撰寫時間：**2002/10/08
>> 
>> **撰寫內容：**基於我被高層關切而導致的兩次審判，在這裡感謝為我辯護的倫理委員。
>> 
>> 自然產生的東西就像是野果，摘採野果以果腹並沒有什麼問題，這甚至是人類最早的求生手段，但是人工產生的東西是需要成本的，也就是代價，打造一把利劍所需要的原料也不必我多說了，如果你希望人工的東西可以擁有除了他應該擁有的性質以外的能力，像是名氣，那自然會讓代價往上爬。
>> 
>> 這個地點本身沒有問題，有問題的是人，這麼說好了，如果有人知道做了某件事情自己不用付出代價，卻可以得到顯而易見巨大的利益的時候，一個正常且理性的人幾乎沒有理由拒絕，或許知道可能損人會停手，那不知道呢？
>> 
>> 我已經上報很多次了，我們使用SCP-ZH-267-B之後，整個設施的人員名單沒辦法跟實際上存在的人員對上，我們總共使用了二十七次，根據歷史資訊可以發現我們至少出現了一百多位人員的落差，這不必我多說，是個正常的基金會員工都能夠推理出發生什麼事情。光是那個令人不安的佛像，我覺得一切就說得通了。
>> 
>> 那今天有幾位三垣對我的作法有意見，沒關係，我當然接受，這兩次的審判也證明了我的清白，以及我對工作的努力與重視，這種代價我們玩不起，民異署都可以理解的事情，基金會自然也該明白。
>> 
>> 不要以為不知道了，你就可以對他們為所欲為。
>>

> = **【附錄：一切裁定理由書】**
> 
>> 
>> **撰寫者：**Dr. Ming
>> 
>> **撰寫時間：**2022/10/08
>> 
>> **前言：**自2008年至今，總共有十八位倫理委員對絕大部分Dr. Ming所做出的有關SCP-ZH-267裁定聲請暫時處分，並在這期間發動兩次四級權限合審，以及兩次五級權限合審，本理由書即於第二次五級權限合審所寫。
>> 
>> **撰寫內容：**首先先感謝為我辯護的大部分五級權限成員以及倫理委員。
>> 
>> 歷年十八位倫理委員對我行為的批評，多為我「犯倫理上嚴重錯誤」、「有義務不如此行為」，或是「效益誠非首重」等等，諸如此類。對於這種批評我是接受的，畢竟倫理委員會的特色在於每位倫理委員都擁有自己的倫理觀念。
>> 
>> 當我們研究者可以在站點內好好做研究，其實就是仰賴於其他保護我們的特工，還有那些高科技防護措施與其製造人，當我們成天做研究，全然不知這些不在眼前的是我們的靠山，於是口出惡言嘲諷對方，我想這是荒謬，而且無知的。我們如果能夠高枕無憂，那就代表有人在負重前行，高枕無憂不是原罪，但請謹記這份安逸的由來。
>> 
>> 我們也當然知道，無論是在保護或值勤時殉職的特工，還是科技開發的黑暗面，例如不人道的實驗或是被作錯誤的用途，這些時候，決策者或開發者都需要承擔做出決定的責任。基金會在面對異常時做的研究與裁定也常常會涉及類似的爭議，而決策者也應該要擔起相對的責任，所以我接受每一次的合審，也接受每一次裁定的異議。
>> 
>> 但在我接受這種責任的同時，我希望各位了解，如果沒有人做成這樣的裁定，Site-ZH-16將在2018年就因為多項災厄滅站並順帶帷幕破裂，而基金會最好的結局也是在2021年因為內亂而解散，SCP-ZH-267的重要性不言可喻。
>> 
>> 我知道我在眾人眼裡呈現的樣子是為了效益而不擇手段的人，我的確很重視效益，但談效益的話，事情不能只看一半。十八位倫理委員曾舉陳默廉 博士與我大相逕庭的行為方針作為擋箭牌，不過我得很誠實地說，陳默廉 博士與我主事的年代和環境截然不同，我們兩個人所信奉的道德教條也不同，但我們的行事是類似的。
>> 
>> 當我們將過往較為和平的年代所做出的裁定按照比例換算在較為凶險的年代，我在這份理由書撰寫的當下，應該做出了超過一百零八次的裁定，這甚至不包括程序較鬆綁時代的數量，而我只做出十八次進入SCP-ZH-267-B的裁定。實際上，在當年還不了解SCP-ZH-267-B時，我們很常將人送進其中製造與取得SCP-ZH-267，基金會從上到下都適應了這種行為，如果無理由地停止，我想陳默廉 博士應該沒辦法順利退休，但他找到了原因，也尋得了理由，我們才能夠將已經麻痺的基金會拉回來。
>> 
>> 對於陳默廉 博士，他無法接受這樣子的代價，無論是做什麼，所以他將SCP-ZH-267-B封鎖；我則是接受為了更大的利益而需要犧牲一些東西，但我認為一個對重要資源犧牲麻痺的基金會，不會是一個最符合效益的樣態，正當的犧牲被濫用，就像是拿菜刀殺人一樣。很高興我們已經從會被施壓派人進入SCP-ZH-267-B的環境轉變，基金會已經重新恢復知覺，我甚感欣慰。
>> 
>> 我的行為在部分倫理教條下的確是錯誤的，我接受非難，也接受審判，這是我選擇的做法所要負擔的代價，我不會閃避。我做出這樣的行為是我接受為了重要利益，我們可以犧牲一些重要的東西，同時我在做出裁定的當下，我會評估這份利益的重大性，使這種手段不被濫用，因為比起濫用這項手段的基金會，不濫用的基金會帶來的效益明顯更大。
>> 
>> 從隨意進入到需要項目研究主任做出裁定，從無處救濟到倫理委員主動追查，程序朝向嚴謹的變化就恰恰反映了這類裁定的茲事體大，我認為大部分的正常人並不會同意我做出這類裁定，而這麼強力的程序保護，其實就是為了讓利益與代價的平衡達到合理的狀態，而不會出現濫用而導致悲劇。
>> 
>> 這種裁定不該是常態，但我得做出抉擇，無論這份抉擇如何被爭論，我都會親上火線正面回應，而不是自詡救世主而傲慢，要眾人感恩我的大恩大德。
>>
